# Анализ отношения к коррупции на данных World Values Survey (WVS Wave 7)

## Описание проекта

Проект посвящён исследованию связи между постматериалистическими ценностями, отношением к коррупции и гендерными различиями на данных **World Values Survey (Wave 7, 2017–2022)**.

Исследование состоит из двух частей:
1. **Основной анализ:** взаимосвязь между индексом постматериализма и отношением к коррупции.
2. **Дополнительный анализ:** гендерные различия в отношении к коррупции.

## Цели исследования
* Проверить, существует ли зависимость между уровнем постматериалистических ценностей и отношением респондентов к коррупции.
* Оценить гендерные различия в оценке допустимости коррупции.
* Провести статистические проверки, визуализировать результаты и оценить размер эффекта и мощность тестов.

## Теоретическая база

Проект основан на:
* Теории постматериализма Р. Инглхарта.
* Гипотезах **Scarcity** и **Socialization**.
* Работах о связи постматериалистических ценностей и коррупционных установок.

**Теоретическая гипотеза:** постматериалисты должны быть менее толерантны к коррупции.
**Эмпирическая гипотеза:** более высокий индекс постматериализма → более низкая одобряемость коррупции.

## Используемый стек
**Язык:** R: `tidyverse` — обработка данных, `ggplot2` — визуализации, `e1071` — асимметрия и эксцесс, `pwr` — статистическая мощность

## Предобработка данных

Основные шаги:
* загрузка данных (`read_csv`)
* фильтрация технических значений (-1, -2, -3, -4, -5)
* переименование переменных
* перекодировка вопросов для построения индекса постматериализма
* создание новой переменной **Index_PM**
* построение итоговых датасетов для основного и дополнительного анализа

```r
wvs_d <- wvs %>% 
  select(Q181, Q152, Q153, Q154, Q155, Q156, Q157) %>%
  filter(if_all(everything(), ~ !(.x %in% c(-1, -2, -3, -5, -4)))) %>%
  rename(Corruption = Q181) %>%
  mutate(
    Q152 = case_when(Q152 %in% c(1, 2) ~ 0, Q152 %in% c(3, 4) ~ 1),
    Q153 = case_when(Q153 %in% c(1, 2) ~ 0, Q153 %in% c(3, 4) ~ 0.5),
    across(c(Q154, Q156), ~ case_when(.x %in% c(1, 3) ~ 0, .x %in% c(2, 4) ~ 1)),
    across(c(Q155, Q157), ~ case_when(.x %in% c(1, 3) ~ 0, .x %in% c(2, 4) ~ 0.5))
  ) %>%
  mutate(Index_PM = Q152 + Q153 + Q154 + Q155 + Q156 + Q157) %>%
  select(Corruption, Index_PM)
```

## Описательная статистика

Вычислены:

* минимум, максимум
* среднее, медиана
* стандартное отклонение
* межквартильный размах
* асимметрия и эксцесс

## Основной анализ: корреляция Спирмена

```r
cor.test(wvs_d$Corruption, wvs_d$Index_PM, method = 'spearman')
```
Результаты:
* **r = 0.023** — крайне слабая положительная зависимость
* **p-value < 0.001** — значимо из-за огромного размера выборки
* **Вывод:** гипотеза не подтвердилась, практически связи нет

Визуализаця (heatmap):
```r
heatmap_data <- table(wvs_d$Index_PM, wvs_d$Corruption)
heatmap_df <- as.data.frame(as.matrix(heatmap_data))
```

## Статистическая мощность

```r
pwr.r.test(n = 86697, r = 0.023)
```
Мощность ≈ **0.99**
Даже крайне слабые связи становятся значимыми при N > 80 000.

# Дополнительный анализ: гендерные различия

**Метод:** тест Манна–Уитни
**Гипотеза:** женщины менее терпимы к коррупции.

```r
wilcox.test(women, men, alternative = "less")
```
Результаты:
* **p-value < 0.001**
* **размер эффекта r = 0.022** (очень слабый)

**Вывод:**
Статистически значимо, но практически — нет.

Визуализация гендерных различий:

# Ограничения исследования
* столь многомерный контрукт требует построения более сложных линейных моделей, так как могут присутствовать скрытые переменные (доход, доверие к институтам, политический контекст)
* межстрановые различия сглаживают эффект
* огромная выборка создаёт ложную значимость микроскопических эффектов

# Итоги проекта
* Связи между постматериалистическими ценностями и отношением к коррупции **практически нет**.
* Женщины действительно менее терпимы к коррупции, но разница **минимальна**.
* Реализованы полный цикл анализа: предобработка → описательная статистика → визуализации → тестирование гипотез → интерпретация.
