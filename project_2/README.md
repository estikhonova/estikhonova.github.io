#  «Анализ данных об оптовых продажах аудиотехники»

**Контекст:** Компания занимается оптовой продажей аудиотехники (профессиональные аудиосистемы, домашние кинотеатры, портативные устройства) и сотрудничает с крупными розничными сетями в России.

**Задача:** В период временной недоступности основного хранилища данных необходимо было собрать информацию из разрозненных резервных выгрузок, провести комплексный анализ продаж и предоставить руководству выводы для принятия тактических решений по управлению ассортиментом и клиентской базой.

**Стек:** 
- Python: Pandas, NumPy (обработка и анализ данных), Matplotlib, Seaborn (представление результатов), Os (сбор информации из системных выгрузок)
- Jupiter

### Этапы работы

**1. Сбор и объединение данных**
- Извлекла данные из иерархической файловой структуры: папки по датам → менеджерам → CSV-файлы
- Объединила информацию о заказах (`orders.csv`), их статусах (`order_status.csv`) и товарах (`products.csv`)
- Применила данные из стороннего текстового файла `usd_rate.txt` для конвертации валют

**2. Очистка и подготовка данных**
- Стандартизировал названия товаров (извлекла бренды и модели из строк)
- Привел данные о датах к единому формату
- Обработал дубликаты заказов (учет отмененных и подтвержденных версий)

**3. Анализ ключевых показателей**
- Рассчитал общую выручку, средний чек, количество подтвержденных заказов
- Проанализировал динамику продаж по периодам
- Определила наиболее прибыльные товарные категории и бренды
- Выявила сегменты клиентов по объему закупок

**4. Исследование причин отмен заказов**
- Сопоставила отмененные и подтвержденные заказы
- Проанализировала, какие товары чаще отменяются
- Исследовала влияние менеджеров на конверсию заказов

### Метрики:
Выручка: 155.4 млн ₽
Средний чек: 503 тыс. ₽
Доля отмен: 11%

### Инсайты:
- CRM сбой - 79% повторных заказов 14 марта после массовых отмен 13 марта
- Недельная цикличность - минимальные продажи по выходным
- Проблемные бренды - 4 бренда с >50% невостребованных товаров
- Эффективность менеджеров - разрыв между лучшим и худшим >15 раз
