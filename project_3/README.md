# Анализ связей базовых ценностей по Ш. Шварцу с чертами личности из модели НЕХАСО 

### Стек:
**Язык:** R 

**Пакеты:**
- `car` - для работы с линейными моделями
- `psych` - продвинутые описательные статистики
- `lmtest` - для дополнительных диагностических тестов
- `effectsize` - размер эффекта
- `sjPlot`, `gglm`, `ggplot2` - диагностические визуализации и графики модели
