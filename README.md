## Проекты

### Проект 1: Разработка и валидизация опросника для измерения левого авторитаризма

**Цель проекта:** Разработать и эмпирически проверить новый русскоязычный инструмент для измерения левого авторитаризма — социально-психологического конструкта, почти не изученного в российском контексте.

**Инструменты:** R (tidyverse, dplyr, ggplot2, lavaan, semTools, semPlot), Excel, статистический анализ (регрессия, корреляции, факторный анализ, проверка надёжности) 

**Этапы работы:**
- Провела глубокий обзор литературы по теме авторитаризма, выделила теоретические компоненты конструкта, на основе которого сформировала базу вопросов для методики.
- Сформировала дизайн исследования и структуру переменных
- Организовала сбор данных на русскоязычной выборке
- Подготовила датасет к анализу в Excel: очистка, проверка выбросов, реверсирование шкал, создание композитных индексов.
- Выполнила статистический анализ в R: оценку внутренней согласованности (α-Кронбаха), корреляции для проверки внешней валидности, регрессионные модели для проверки гипотез, факторный анализ для проверки структуры шкалы
- Построила визуализации и подготовила аналитический отчёт по результатам.

**Результат:**
Проект привёл к созданию первой версии русскоязычного инструмента для измерения левого авторитаризма. Шкала продемонстрировала приемлемую надёжность и ожидаемые связи с внешними переменными. Результаты легли в основу курсовой и были представлены на заседании лаборатории политико-психологических исследований НИУ ВШЭ.
